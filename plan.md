# Plan — Site Web Photographe Professionnel

## Objectif
Créer un site web d'une beauté exceptionnelle pour un photographe professionnel avec galerie portfolio et boutique d'images.

## Stages

### Stage 1 — Design & Setup
- Charger `vibecoding-webapp-swarm` skill
- Concevoir l'architecture visuelle : hero immersif, galeries, boutique, à propos, contact
- Générer les assets visuels (images du photographe, photos de démonstration)

### Stage 2 — Développement
- React + TypeScript + Tailwind CSS + shadcn/ui
- Animations fluides (Framer Motion)
- Sections : Hero, Portfolio/Galerie, Boutique (vente d'images), À propos, Contact
- Navigation élégante avec transitions de page
- Panier d'achat pour la boutique d'images
- Design responsive

### Stage 3 — Déploiement
- Build et déploiement du site

## Livrable
- Site web déployé, URL publique
